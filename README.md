# website

This project is generated by Grace v2023.0.0.

### Versions

* Grace 2023.0.0
* Groovy 4.0.22
* Spring Boot 3.0.13
* Spring Framework 6.0.23
* Tomcat 10.1.28

```
create-app website
cd website
create-domain-class Sample
git checkout app/domain/website/Sample.groovy
generate-controller Sample
generate-views Sample
install-templates
```
